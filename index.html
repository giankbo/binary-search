How does it work?

The project first does a random search for the names: 'Iron Man', 'Captain America', 'Thor', 'Hulk', 'Black Widow', 'Hawkeye'. Then, it locates the first letter of the name and does a linear and binary search for that letter in the alphabet. Finally, it shows the number of interactions for each search.
